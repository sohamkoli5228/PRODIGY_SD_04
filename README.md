# Sudoku Solver

A Python application that solves 9×9 Sudoku puzzles using the Backtracking Algorithm.

## Features

- Solve Sudoku puzzles
- Backtracking Algorithm
- Original and Solved Board Display
- Input Validation
- Menu-Driven Interface

## Technologies Used

- Python 3
